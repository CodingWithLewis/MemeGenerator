## Getting Started

The app runs on Streamio and mostly API endpoints.

1. Create a `.env` file in root.
2. Insert relevant API Keys:

```
ANTHROPIC_API_KEY=

OPENAI_API_KEY=

NEON_POSTGRES=

COHERE_API_KEY=
```

3. build the docker container
